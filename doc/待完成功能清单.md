# 海外版的规划
## 发展前景估计
基于几个点：

1、国外表情制作这个细分领域是存在的，google play搜索可得

2、我们的产品在国外的这个细分领域，从技术上应该是有足够竞争力的，原因：
2-1、当前我们的产品在国内在这个细分领域是有竞争力的，当然不是我们多厉害，是因为哪些厉害的都不来做这个
2-2、很多情况下，现在在国内做得好的应用出海在国外实际上也有一定竞争力的，比如最厉害的抖音，原神
2-3、然后直观感觉上，从用户体验和功能丰富度上来说我们的应该要比国外的一些产品好不少
综上，我们的产品技术上在国外应该是有一定竞争力的

结论：领域存在，我们在领域技术上具有竞争力，那么我们做好内容和运营，外加自身努力，就能在国外应该能取得好的效果

## 风险，不太确定的是：
